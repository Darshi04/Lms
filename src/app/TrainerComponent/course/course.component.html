Core Java

Core Java covers the fundamentals of Java programming language:

  • Introduction to Java & its history
  • Variables, Data Types, and Operators
  • Control Flow Statements (if, else, switch)
  • Object-Oriented Programming Concepts (OOP)
  • Exception Handling and Debugging
  • Java Collections Framework
  • Working with Files, Streams, and Serialization

Duration: 2 weeks | Level: Intermediate to Advanced

Data Structures

This course covers essential data structures and algorithms:

  • Arrays, Linked Lists, and Stacks
  • Queues, Heaps, and Hash Tables
  • Binary Trees, Binary Search Trees
  • Graphs and Graph Traversal (BFS, DFS)
  • Sorting Algorithms (Bubble, Merge, Quick)
  • Time Complexity and Big O Notation

Duration: 4 days | Level: Intermediate

Cloud Computing

In this course, you will learn about cloud computing concepts and platforms:

  • Cloud Platforms: AWS, Google Cloud, Microsoft Azure
  • Virtualization & Cloud Architecture
  • Cloud Storage, Networking, and Security
  • Deploying and Managing Cloud Applications

Duration: 4 days | Level: Intermediate

Linux

  • Linux Command Line and Shell Scripting
  • Linux File System and Permissions
  • Package Management and System Maintenance
  • Networking and Security Basics on Linux

Duration: 3 days | Level: Beginner to Intermediate
